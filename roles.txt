Роли в приложении:

Админ - единственный, кому доступна админка и все функции на сайте.

Контент-менеджер - человек, который может писать, редактировать, удалять контент.
Доступ в админку и личные кабинеты пациентов запрещен.
Назначается админом из числа пациентов

Психотерапевт - зарегистрированный пользователь, имеет доступ к личному кабинему пациента
Имеет в своем личном кабинете список пациентов.
Назначается админом из числа пациентов

Пациент - имеет доступ только к своему личному кабинету, может читать статьи
При регистрации любой пользователь становится пациентом.

-- Создание таблицы role
CREATE TABLE role (
    role_id INT PRIMARY KEY,
    role_name VARCHAR(50) NOT NULL
);

-- Создание таблицы permission
CREATE TABLE permission (
    permission_id INT PRIMARY KEY,
    permission_name VARCHAR(50) NOT NULL
);

-- Создание таблицы для связи между role и permission
CREATE TABLE role_permission (
    role_id INT,
    permission_id INT,
    FOREIGN KEY (role_id) REFERENCES role(role_id),
    FOREIGN KEY (permission_id) REFERENCES permission(permission_id),
    PRIMARY KEY (role_id, permission_id)
);
